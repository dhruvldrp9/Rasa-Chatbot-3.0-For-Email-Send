# Rasa-Chatbot-3.0-For-Email-Send
##This is an automatic mail send chatbot with rasa 3.0
## create virtual environment
```
python3 -m venv venv
```
## activate virtual environment
```
source venv/bin/activate
```
## Install rasa3.0
```
pip install rasa
```
## create initial bot
```
rasa init --no-prompt
```
## Replace all file 
## For trainig
```
rasa train
```
## For test
```
rasa run actions
```
##open another tab
```
rasa shell
```
